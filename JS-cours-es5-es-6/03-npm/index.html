Node Package Manager

NPM permet de gérer les dépendances de votre projet en ajoutant/supprimant des packages en 1 ligne de commande depuis le terminal.

Les commandes courantes

  • npm install nomDuPackage
    pour installer un package
  • npm upinstall nomDuPackage
    pour updater un package
  • npm uninstall nomDuPackage
    pour désinstaller un package

Exemple ici, téléchargeons jQuery dans notre dossier de projet

  1. Se placer dans le dossier en ligne de commande (depuis le terminal)
  2. Taper la commande npm install jquery

Cette commande va créer :

  • le fichier package.json qui répertorie les packages et versions disponibles dans votre projet
  • le dossier node_modules qui contiendra le package jquery

Vous n'avez plus qu'à l'utiliser ici pour vérifier que jquery est bien disponible, en l'insérant dans votre page.

Vous savez maintenant gérer les dépendances de votre projet en installant/désinstallant des packages avec NPM.
